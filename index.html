الشروط والأحكام

1) يشترط وجود رخصة قيادة سارية وهوية أو جواز سفر ساريين وقت الاستلام.

2) يتم تحديد مبلغ التأمين وفق فئة السيارة ويُعاد حسب سياسة الفحص عند التسليم.

3) الالتزام بمواعيد التسليم والاستلام لتجنب أي رسوم إضافية وفق العقد.

4) يتحمل المستأجر المخالفات المرورية وأي أضرار ناتجة عن سوء الاستخدام.
